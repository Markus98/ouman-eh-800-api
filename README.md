# ouman-eh-800-api
Python library for communicating with the [Ouman EH-800 heating controller](https://ouman.fi/en/product/ouman-eh-800-and-eh-800b/). Primarily designed to be used in the Ouman EH-800 Home Assistant integration.

## Disclaimers
This client has been developed and tested with the Ouman EH-800 unit that was available for the developer. It did not have all features enabled, thus there may be missing features or bugs. Pull requests with new covered features or bugfixes are welcome.
